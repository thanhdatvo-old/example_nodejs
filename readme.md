## II. Installation
#### 1. Front-end

- Install [Angular 2 Cli][angularcli]
- Go to public folder
    ```
    $ cd public
    ```
- Run angular 2 cli the front-end run at gate [4200][4200]

    ```
    $ ng serve
    ```

#### 2. Back-end

- At root folder of project, run

    ```
    $ node app
    ```
